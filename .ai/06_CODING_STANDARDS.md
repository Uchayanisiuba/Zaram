
# Coding Standards

## Principles

- Readability over cleverness
- Composition over inheritance
- Runtime isolation
- Strong typing
- No circular dependencies

## Python

- Type hints
- Dataclasses where appropriate
- Async where beneficial

## React

- Functional components
- Hooks only
- Zustand state
- Tailwind
- Framer Motion

## Naming

Runtime_Name

Service_Name

Interface_Name

Never abbreviate names.
