# Architecture Changelog v1.0

## v1.0 (Current)
- **Added:** Electron Desktop Architecture.
- **Added:** Presence Runtime as permanent embodiment gateway.
- **Added:** Animation Runtime with Emotion Engine and Living Rhythm.
- **Changed:** FrameState frozen with nested namespaces (visual, audio, emotion, system, metadata).
- **Changed:** Migrated from browser-only to Electron-hosted GPU-first rendering.
- **Deprecated:** Direct application-to-renderer communication.

## v0.9 (Previous)
- Initial Runtime Contract definition.
- Event Bus specification.
- Browser-based Living Orb implementation.
