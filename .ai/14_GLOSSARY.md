# Glossary
**Version:** 1.0 | **Status:** Frozen

- **Runtime:** An independent subsystem capable of lifecycle management.
- **FrameState:** The canonical data structure driving visual embodiments.
- **Presence Runtime:** The abstraction layer between the OS and renderers.
- **Strangler Fig:** A migration pattern where new systems are built alongside legacy systems until the legacy system can be safely removed.
- **IEmbodiment:** The interface contract all visual renderers must implement.
- **Kernel:** The core orchestration layer (Registry, Event Bus, Bootstrapper).
