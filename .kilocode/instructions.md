# Kilo Code Specific Instructions

When executing multi-file implementation tasks:

1. Always reference `.ai/05_EXECUTION_ENGINE.md` and `.ai/02_RUNTIME_CONTRACT.md`.
2. Use the templates located in `backend/templates/` as your starting point for new Runtimes.
3. Ensure every new file has a corresponding test file in `backend/tests/`.
4. Do not modify files in `.ai/` or `backend/core/` unless explicitly instructed by the user.
